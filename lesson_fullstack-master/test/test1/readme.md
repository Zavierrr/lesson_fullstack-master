- RegExp 正则表达式 对象  
    Reg 规则  
    Exp Expression 表达  
    / /  [] 一个单元 {}  匹配次数（即长度）
    ^ 开始符 会匹配到以指定内容开头的字段
        例：/^a/ 即会匹配到 a结尾的字段    12abbb3ca  得到abbb   
    $ 结束符 会匹配到以指定内容结尾的字段
        例：/a$/ 即会匹配到 a结尾的字段    12abbb3ca  得到ca
