#   npm
- npm run dev       
    dev -> development  开发
- npm run build
    得到dist文件夹    
    编译 打包送到云服务器
- 上线  
    买台服务器（阿里云、腾讯云）
    用户通过公有IP访问
##  流程
    dev（写代码，本地3000调试）-> build（打个包）-> 上线（云服务器IP）
