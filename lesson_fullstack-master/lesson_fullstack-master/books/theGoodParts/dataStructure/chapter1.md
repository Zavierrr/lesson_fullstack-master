#   数据结构（data structure）
        - 是数据的组织、管理 和存储格式，其使用目的是为了高效地访问和修改数据。
        - 数据结构是算法的基石
##  数据结构的组成方式（分类）
        1. 线性结构
            - 最简单的数据结构
                包括数组、链表，以及由它们衍生出来的栈、队列、哈希表。
        2. 树
            - 相对复杂的数据结构 
                包括二叉树以及衍生出来的二叉堆等。
        3. 图
            - 更为复杂的数据结构
                在图中会呈现出多对多的关联关系。
        4. 其他数据结构
            - 由基本数据结构变形而来，用于解决某些特定问题
                包括跳表、哈希链表、位图等
